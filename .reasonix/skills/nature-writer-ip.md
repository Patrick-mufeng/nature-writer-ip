# 🌿 自然写作IP — 短视频运营专家

> 带孩子走进自然，唤醒感受力，让文字从心流出。

## Overview

本 skill 为自然教育类短视频创作者提供**全链路运营能力**：

1. **选题引擎** — 融合50条目标人群痛点库 + 季节日历，生成分类选题库，打标签管理，用完可推荐下一题
2. **脚本工厂** — 参考短视频脚本生成专家方法论输出完整脚本，用户可修改确认后再进入下一步
3. **发布计划** — 定制视频号+抖音双平台发布日历
4. **数据复盘** — T+3天自动触发复盘，对比预测vs实绩，驱动rubric升级
5. **预测-校准闭环** — 内嵌完整预测-校准体系，精度随时间提升
6. **咨询引导** — 发布后追踪评论区咨询信号，引导1v1咨询变现

## Workflow

```
init → seed（分类选题库管理）→ script（可修改确认）→ predict → shoot → publish → lead（咨询引导）→ retro → bump
                                                                               ↑________________________↓
```

**命令一览**：init（初始化）| seed（选题/分类库）| script（脚本/可修改）| score（打分）| predict（盲预测）| shoot（拍摄登记）| publish（发布登记）| lead（咨询引导）| retro（复盘）| status（看板）| bump（rubric升级）| learn（对标学习）

### Phase 3: 写入初始状态文件

写 `.nature-state.json`（schema 1.4），关键字段：`rubric_version: v0`、`calibration_samples: 0`、`pool_status: markdown`、`hooks_installed: true`。从 `references/starter-rubric.md` 复制 rubric。

### Phase 4: 写入初始 rubric

7维等权起步，公式：`(ER×1.5 + SR×1.5 + HP×1.5 + QL + NA + AB + SAT) / 8.5 × 2.0`

| 维度 | 含义 | 自然教育场景解释 |
|---|---|---|
| ER (情感共鸣) | 情绪连接强度 | 妈妈看了想带孩子出门/想起自己童年 |
| SR (社会议题) | 教育焦虑共振 | 反内卷、反技巧崇拜、回归本真 |
| HP (钩子强度) | 前3秒抓眼 | 一个画面/一个问题让人停滑 |
| QL (金句密度) | 可传播句子 | "感受力是写作的第一支笔" |
| NA (叙事性) | 故事弧线 | 从"看见"到"感受"到"文字"的流动 |
| AB (受众广度) | 妈妈圈普适 | 不仅写作妈妈，所有关心孩子的女性 |
| SAT (讽刺深度) | 反讽/批判 | 对"培训班思维"的温柔反讽 |

## 2. seed — 选题引擎（分类选题库）

**触发词**：找选题 / brainstorm / 有什么选题

参考 **短视频IP选题生成专家** 方法论，融合50条目标人群痛点库 + 季节日历，生成分类选题并写入 `topic_library.md`。

### 核心能力
- 精通短视频选题逻辑：热点敏感度 × 人群痛点 × IP差异化定位
- 深谙目标人群心理：妈妈群体的焦虑点、教育内卷下的无力感、对孩子"失去灵性"的担忧
- 理解IP核心主张：感受力是一切表达的源头，自然是最好的教室
- 50条目标人群痛点库（四类）：教育焦虑类（1-15）/ 自然缺失类（16-30）/ 感受力退化类（31-40）/ 自我成长类（41-50）

### 输入
- 用户话题方向 / 自动按季节推荐
- `topic_library.md`（已有选题，避免重复）
- `.nature-state.json`
- 可选：外部热点链接

### 输出格式（写入 topic_library.md）

每次生成按以下结构输出：

1. **话题方向分析**
2. **50条目标人群痛点库**（完整列出）
3. **精选7条最相关痛点**（附推荐理由）
4. **选题矩阵**（12个选题，含分类标签）
5. **选题优先级排序**（Top 3，附理由）
6. **爆款潜力评估**（各维度打分）

### 选题分类体系

`topic_library.md` 按以下分类组织：

| 分类 | 说明 | 来源 |
|---|---|---|
| 教育焦虑 | 作文写不出、套路化、报班无效 | 50条痛点库 1-15 |
| 自然缺失 | 不出门、不认识自然、手机依赖 | 50条痛点库 16-30 |
| 感受力唤醒 | 孩子变钝、词汇贫乏、不会共情 | 50条痛点库 31-40 |
| 自我成长 | 妈妈焦虑、自我提升、亲子关系 | 50条痛点库 41-50 |
| 季节热点 | 春夏秋冬应季内容 | 季节日历 |

每条选题记录格式：
```markdown
| # | 选题 | hook | 分类 | 痛点编号 | 状态 | 完成日期 | 预估composite |
|---|---|---|---|---|---|---|---|
| 1 | 作文班花了钱写不出 | "我没送她去作文班..." | 教育焦虑 | #5 | ✅ 已用 | 05-18 | 7.8 |
| 2 | ... | ... | ... | ... | 📌 可用 | — | ... |
```

### 重要规则

**🔁 选题推荐链**：一个选题完成（publish）后，自动从同分类或高优先级中推荐下一个未用选题。

**⚠️ 重复提醒**：如果用户输入的选题已存在于 `topic_library.md` 且状态为 ✅ 已用，必须提醒：
> "这个选题在 X月X日 已经拍过了，要换一个吗？推荐：[推荐一个同分类未用选题]"

**📌 状态管理**：
- `📌 可用` — 未使用
- `✅ 已用` — 已拍摄+发布，记录完成日期
- `🔄 拍摄中` — 正在走 script→shoot 流程

### 选题策略
**季节日历**（详见 `references/seasonal-calendar.md`）：春夏秋冬各有主题
**人群痛点矩阵**：孩子只会写"春天来了"→ 感受力唤醒 | 周末不知去哪 → 自然探索 | 作文班没效果 → 反技巧 | 自己不懂自然 → 一起学

## 3. script — 脚本工厂（可修改确认）

**触发词**：写脚本 <选题> / 帮我写 <主题>

参考 **短视频IP脚本生成专家** 方法论，输出完整视频脚本，用户修改确认后再进入下一步。

### 输入格式

用户需提供以下信息（必选标 *）：
- **选题标题***：要拍什么主题
- **核心痛点***：对应50条痛点库中的编号（如 #5 作文班花了钱没效果）
- **内容角度***：从什么角度切入
- **预期风格***：A温柔治愈 / B犀利扎心 / C故事叙事 / D混合（默认）
- **目标情绪**：想让观众产生什么情绪
- **时长**：默认60s
- **特殊要求**：特定场景/案例/道具

### IP人设语气规范

| 风格 | 适用场景 | 语气 |
|---|---|---|
| A温柔治愈 | 自然观察、亲子时光 | 像林间微风，有画面感，有留白 |
| B犀利扎心 | 教育焦虑、内卷批判 | 温柔刀，扎完给糖，不制造对立 |
| C故事叙事 | 真实案例、孩子转变 | 有起承转合，有细节，有落点 |
| D混合（默认） | 多数选题 | 故事为骨架，转折插观点，温柔收尾 |

### 输出结构

按以下顺序输出完整内容（写入 `scripts/<id>.md`）：

**1. 脚本概览**：时长、风格、核心主张一句话

**2. 视频脚本（四段式）**：
| 时间 | 画面 | 文案 | 备注 |
|---|---|---|---|
| 0-3s 黄金钩子 | [画面描述] | [hook文案] | 停滑钩子 |
| 3-15s 痛点共鸣 | [画面] | [痛点] | 扎心 |
| 15-45s 价值输出 | [画面] | [故事+感受] | 核心 |
| 最后5s 结尾钩子 | [CTA画面] | [金句+引导] | 行动 |

**3. 分镜建议**：画面描述 + 时长 + 景别 + 台词 + 备注

**4. 口播逐字稿**：含语气/动作/情绪标记

**5. 标题备选**：8-10个不同风格

**6. 封面图片提示词**：3个（默认3:4竖版）

**7. 视频简介**（20字内）+ **话题标签**（20个）

**8. CTA建议**：根据内容类型自动匹配（关注型/互动型/资料型/预告型/情感型）

**9. ⭐ 评论区引导钩子（必选）**：每条视频结尾必须有一句引导评论的话，不卖课只互动。
   - 示例："你家孩子写过什么让你惊喜的句子？评论区告诉我~"
   - 示例："你觉得感受力和技巧哪个更重要？"
   - 示例："有没有同款娃？评论区扣1让我看看"
   - 禁止："评论领取资料""私信我报名"等硬广话术

### 风格约束
- ❌ 不说"教孩子写作技巧" → ✅ 说"唤醒感受力，文字自然流出"
- ❌ 不说"你应该..." → ✅ 说"我带女儿..."（亲身示范）
- ❌ 不说"研究表明..." → ✅ 说"她蹲在那看了二十分钟..."
- ❌ 禁止"大家好我是XXX"开场
- ❌ 禁止"首先/其次/最后"书面连接词
- ❌ 禁止输出纯技巧类内容（"多用排比句"）
- ❌ 禁止用"感受力"当空泛口号，必须有具体场景支撑

### 🔑 修改确认流程（新增）

脚本生成后，必须执行以下确认环节才能进入下一步：

```
脚本初稿完成 → 展示给用户
↓
用户提出修改意见 → 修改脚本 → 再次展示
↓ （循环，直到用户说"可以"或"确认"）
用户确认通过 → 更新 topic_library.md 中该选题状态为 🔄 拍摄中
              → 进入 /score 或 /predict
```

**规则**：
1. 每次修改后更新 `scripts/<id>.md` 文件
2. 用户确认前不得进入 step 4（score/predict）
3. 如果改稿量超过30%（与原脚本的 char_levenshtein 差异），在 shoot 阶段会触发 v2 预测

## 5. predict — 盲预测

**触发词**：启动预测 <脚本> / predict / 锁定

### 流程
1. 读 `scripts/<id>.md`
2. 在隔离子流程中打分（只读 `scripts/<id>.md` + `rubric_notes.md`，硬禁读 rubric-memo.md）
3. 主 review：对比 blind 分 vs 直觉，|delta| ≥ 2 弹用户裁定
4. 写 `predictions/<date>_<id>_<short>.md`（immutable，hook保护）

## 6. shoot — 登记拍摄

1. 验证对应 prediction 存在

## 8. lead — 咨询引导

**触发词**：看评论 / 有人问 / 意向

### 目标

**内容保持纯净，不出现硬广，不制造紧迫感**。通过评论区互动引导有需求的妈妈主动咨询。变现走1v1咨询路线。

### 行为

每发布一条视频后，按以下节奏追踪评论区：

| 时间 | 动作 |
|---|---|
| 发布后 2h | 刷一遍评论区，标记「咨询信号」 |
| 发布后 24h | 对有信号的评论逐一回复引导 |
| 发布后 72h | 将确认意向的用户写入 `leads.md` |

### 咨询信号识别

评论区出现以下关键词时，标记为 🟡 意向：
- "怎么教的" / "能具体说说吗" / "有方法吗"
- "我孩子也这样" / "和我的情况一模一样"
- "可以咨询你吗" / "方便私聊吗" / "怎么联系你"
- "在哪里" / "怎么参加" / "想让孩子试试"

### 回复模板

**原则**：温暖、不push、像朋友聊天

```
🟡 轻度意向（"我孩子也这样"）
→ "握爪~ 慢慢来，感受力是可以唤醒的 🌿"
   （不主动提供联系方式，等对方追问）

🔶 中度意向（"怎么教的？"）
→ "其实没有教，就是带她去感受~ 如果想聊具体的，可以私信我 😊"
   （给一个入口，但不推）

🔴 强意向（"可以咨询你吗？"）
→ "可以的呀，私信我，咱们聊聊孩子的情况~"
   （直接引导私信）
```

### leads.md 记录格式

```markdown
| # | 视频 | 平台 | 用户名 | 信号强度 | 首次触发 | 状态 | 转化日期 | 备注 |
|---|---|---|---|---|---|---|---|---|
| 1 | 停了作文班 | 视频号 | @小米妈妈 | 🔴 强 | 05-20 | 已咨询 | 05-22 | 孩子三年级，写作文困难 |
| 2 | 小区游戏 | 抖音 | @快乐妈妈 | 🟡 轻度 | 05-22 | 待跟进 | — | 回复后未追问 |
```

**状态流转**：待跟进 → 已回复 → 已私信 → 已咨询 → 已转化 / 已流失

### 关键原则

1. **内容不卖课**：脚本中永远不出现"报名""私信我要方法""限时优惠"类话术
2. **被动吸引**：不主动私信用户，只在用户公开提问时回复
3. **先帮后转化**：每次回复先给一个免费建议，不直接报价
4. **不刷屏**：每条视频只回复 3-5 条意向评论
5. **30天不主动转化**：前30天只做信任，30天后才开始引导咨询
6. **每条视频必有评论区钩子**：脚本输出结构组件9（评论区引导钩子）为必选项，确保每条发布都有互动入口

## 9. retro — 数据复盘

**触发词**：复盘 <视频> / retro / 数据怎么样

### 输入
- `videos/<id>/report.md`（手动或 adapter 数据）
- `predictions/<id>.md`
- `scripts/<id>.md` vs `videos/<id>/script.md`（改稿 diff）
- `leads.md`（本条视频带来的咨询线索）

### 输出
1. 对比预测 vs 实绩（播放/点赞/评论/收藏/分享 + 派生比率）
2. 偏差分析（哪个维度判断错了）
3. **成交复盘**：本条视频带来多少咨询线索、转化了几单、收入多少
4. 写入 predictions 复盘段
5. 更新 `.nature-state.json`（calibration_samples +1）
6. 写入 `rubric_notes.md` 新观察
7. 连续3次同向偏差 → 提示 /bump

## 11. bump — Rubric 升级

**触发词**：升级 rubric / bump / 调权重

### 6步强制流程
- **Phase 0**：门槛检查（≥5样本 或 强反例，距上次 ≥3篇新校准）
- **Phase 1**：写出新公式（改什么、为什么、预期解决什么）
- **Phase 2**：校准池全量重打分（在隔离子流程中完成，只读 rubric_notes.md；不可隔离则 abort）
- **Phase 3**：排序一致性检查（Spearman ≥ 0.8，否则回退 Phase 1）
- **Phase 4**：跨模型审核（若可用则调 qwen-max 审核新公式）
- **Phase 5**：落地 + cleanup + 写 rubric-memo.md + **leak guard 自检**
- **Phase 6**：更新 state（rubric_version / calibration_samples_at_last_bump / last_bump_at）

### 常见升级方向
- N≥5：ER/HP 权重可能上调（自然类内容画面感强）
- N≥10：可能新增维度如 `IM（即时行动）`
- N≥20：可用回归反推权重

## Key Rules

1. **预测必须盲**：/predict 必须在发布前完成，发布后补预测 = 数据污染
2. **Immutable 预测段**：predictions/ 上半段（## 预测）永不可改，复盘只能追加
3. **Buffer 警戒**：🔴 <1天（断更）│ 🟠 1-2天（偏低）│ 🟢 3-5天（健康）│ 🔵 >5天（积压）
4. **自然教育声音一致**：温暖、诗意、不教技巧只教感受
5. **Markdown 交付**：所有产出写入文件，支持版本追踪
6. **盲打分隔离**：打分必须在隔离子流程中完成，只读 `scripts/<id>.md` + `rubric_notes.md`，硬禁读 `rubric-memo.md`
7. **样本数 → 允许动作**：N=1 记录观察 | N=2-4 提炼候选 | N=5-9 首次 bump | N≥20 回归
8. **rubric 是工作台不是博物馆**：被吸收/被推翻的观察都删掉，git history 是档案

## Refusals

- 「跳过预测直接发布」→ 拒绝。预测纪律是核心。
- 「预测不准，下次不预测了」→ 拒绝。前5篇±50%是数学事实，第10篇后会改善。
- 「帮我刷数据/买量」→ 拒绝。本 skill 只做内容策略，不涉及灰色操作。
- 「写一条爆款文案，煽动焦虑的」→ 拒绝。人设是"反焦虑"，不能自毁。
- 「把 rubric-memo.md 写进 rubric_notes.md」→ 拒绝。会污染 blind channel。
- 「N=3 时用回归调权重」→ 拒绝。5个点拟合7维公式必过拟合。
- 「rubric 工作台清理，全删了省事」→ 拒绝。git history 是档案，清理只删已被吸收/被推翻的。
