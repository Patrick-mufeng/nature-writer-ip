# 🌿 自然写作IP — 短视频运营专家 Skill

> 带孩子走进自然，唤醒感受力，让文字从心流出。

## 简介

**nature-writer-ip** 是一个面向自然教育类短视频创作者的 **Reasonix Skill**，覆盖选题→脚本→预测→拍摄→发布→咨询引导→复盘→rubric升级全链路。

| 项目 | 内容 |
|---|---|
| **人设** | 自然里的写作老师 / 感受力唤醒师 / 不教技巧只教感受 |
| **目标人群** | 20-45岁女性（妈妈、教育工作者） |
| **平台** | 视频号 + 抖音 |
| **运行方式** | Reasonix subagent（`run_as: subagent`） |
| **模型** | deepseek-v4-pro |

---

## 技能架构

由 **1 个主 skill + 2 个子 skill** 组成：

```
nature-writer-ip（主 skill，12 个命令）
├── 调用 → topic-expert（选题生成子 skill）
├── 调用 → script-expert（脚本生成子 skill）
└── 调度其余命令
```

主 skill 按需调用子 skill，子 skill 在隔离子代理中运行。

## 工作流程

### 全流程

```
init → seed → script → predict → shoot → edit → publish → lead → retro → bump
                                                         ↑________________↓
                                             （复盘数据驱动下一轮选题优化）
```

### 12 个命令

| # | 命令 | 归属 | 功能 |
|---|---|---|---|
| 1 | **init** | 基建 | 5问onboarding，创建脚手架文件 |
| 2 | **seed** | 内容 | 调用 `topic-expert` 生成选题 + 痛点积累 + 写入双版本选题库 |
| 3 | **script** | 内容 | 调用 `script-expert` 生成完整脚本，用户修改确认后再进入下一步 |
| 4 | **score** | 内容 | 7维 rubric 快速打分（仅控制台，不写文件） |
| 5 | **predict** | 内容 | 盲预测，写入 immutable 预测文件 |
| 6 | **shoot** | 内容 | 登记拍摄，检测改稿差异 |
| 7 | **edit** | 内容 | 剪辑制作（单人出镜 60s 视频约需 0.5-1 天） |
| 8 | **publish** | 内容 | 登记发布，触发推荐链 |
| 9 | **lead** | 变现 | 评论区咨询信号识别 + 回复引导 + 录入线索 |
| 10 | **retro** | 运营 | T+3 数据复盘（含内容数据 + 成交数据） |
| 11 | **status** | 运营 | 状态看板（buffer + 线索 + 收入） |
| 12 | **bump** | 运营 | Rubric 升级（≥5样本后触发） |
| 13 | **learn** | 运营 | 对标账号学习 |

## 各步骤详解

### ① init — 初始化

一次性配置。回答 5 个问题：

- Q1：内容形态（自然观察 / 写作引导 / 教育理念 / 混合）
- Q2：发布状态（从零开始 / 已发过）
- Q3：发布频率（日更 / 隔日 / 每周2-3次 / 灵活）
- Q4：数据回收（手动 / 适配器）
- Q5：对标账号（现在导入 / 等下 / 不找）

**产出**：`.nature-state.json`、`rubric_notes.md`、`topic_library.md` 脚手架等

### ② seed — 选题

调用 `topic-expert` 子 skill，根据话题方向动态生成痛点 + 选题矩阵。

**流程**：
1. 用户输入话题方向（如"作文相关""亲子关系"）
2. `run_skill({ name: "topic-expert", arguments: "..." })`
3. 子 skill 返回：话题分析 → 动态痛点 → 选题矩阵（12个）→ 优先级排序 → 爆款评估
4. 痛点和选题追加到 `topic_library.md`（详细版）和 `topic_library_clean.md`（精简版）

**痛点积累**：每次生成的痛点存入 `pain_points.md`，长期积累形成专属痛点库。

**状态管理**：📌 可用 → 🔄 拍摄中 → ✅ 已用

**智能规则**：
- 重复选题提醒：用户选已用过的选题时提醒
- 推荐链：一个选题发布后自动推荐下一个

### ③ script — 脚本

调用 `script-expert` 子 skill，按以下结构输出完整脚本：

1. 脚本概览
2. 四段式视频脚本（黄金钩子 → 痛点 → 价值 → 结尾）
3. 分镜建议（~15 个镜头）
4. 口播逐字稿（含语气标记）
5. 标题备选（8-10 个）
6. 封面图片提示词（3 个）
7. 视频简介 + 话题标签
8. CTA 建议
9. ⭐ 评论区引导钩子（必选）

**关键环节**：脚本生成 → 用户修改确认 → 确认后才进入下一步。
**命名规则**：`scripts/<日期>_<选题关键词>.md`

### ④ score — 快速打分（可选）

基于 rubric 7 维对脚本做快速评估，仅控制台输出，不写文件。

### ⑤ predict — 盲预测

**核心纪律**：发布前完成预测，发布后补 = 数据污染 ❌

流程：隔离子流程打分 → 对比直觉 → 分歧≥2 弹用户裁定 → 写入 immutable 预测文件。

**产出**：`predictions/<id>.md`（预测段不可改，复盘段可追加）

### ⑥ shoot — 拍摄登记

验证 prediction 存在 → 登记实际拍摄稿 → 检测与脚本一致性（≥30% 差异触发 v2 预测）

### ⑦ edit — 剪辑

预留给剪辑制作环节。单人出镜 60s 视频约需 0.5-1 天（字幕/配乐/封面/转场）。

### ⑧ publish — 发布登记

出队 → 标记 T+3 待复盘 → 更新日历 → 触发推荐链

### ⑨ lead — 咨询引导（变现入口）

**原则**：内容不卖课，被动吸引，先帮后转化。

评论时间线：发布后 2h 扫信号 → 24h 回复引导 → 72h 录入 `leads.md`

**信号识别**："怎么教的""我孩子也这样""可以咨询你吗"等关键词。

**回复模板**：🟡 轻度不推 / 🔶 中度给入口 / 🔴 强意向引导私信。

**前 30 天只做信任，30 天后才开始引导咨询。**

### ⑩ retro — 数据复盘

T+3 天触发，对比预测 vs 实绩，同时复盘成交数据。

**输出**：内容数据（播放/点赞/收藏/评论/分享） + 成交数据（咨询信号数/转化/收入） + 内容类型×转化率跨视频统计 + Bump 触发评估

### ⑪ status — 状态看板

实时显示：Buffer 状态、发布进度、咨询线索（待跟进/已私信/已咨询/已转化/累计收入）

### ⑫ bump — Rubric 升级

6 步强制流程：门槛检查 → 新公式 → 全量重打分 → 排序一致性 → 跨模型审核（可选） → 落地

### ⑬ learn — 对标账号学习

导入对标账号样本，分析 rubric 特征，更新 benchmark。

## 应急预案

| 场景 | 应对 |
|---|---|
| 今天拍不了 | Buffer≥1 发库存，Buffer=0 发图文空镜，连续3天降频 |
| 临时换选题 | 改回 📌 可用，脚本保留，重新 seed |
| 负面评论 | 不删评、不争论、质疑回一次、攻击零回复 |
| 数据远低预期 | 选题问题降权，制作问题改进，算法问题不动 |
| 断更后重启 | 先发"我回来了"→ 前3条不进 predict/retro → 3条后恢复 |

| 风格 | 适用场景 | 语气 |

## Rubric 评分维度（v0）

7维等权，公式：`(ER×1.5 + SR×1.5 + HP×1.5 + QL + NA + AB + SAT) / 8.5 × 2.0`

| 维度 | 含义 | 自然教育解释 |
|---|---|---|
| ER | 情感共鸣 | 妈妈看了想带孩子出门/想起自己童年 |
| SR | 社会议题 | 反内卷、反技巧崇拜、回归本真 |
| HP | 钩子强度 | 前3秒一个画面/问题让人停滑 |
| QL | 金句密度 | "感受力是写作的第一支笔" |
| NA | 叙事性 | 从"看见"到"感受"到"文字"的流动 |
| AB | 受众广度 | 不仅写作妈妈，所有关心孩子的女性 |
| SAT | 讽刺深度 | 对"培训班思维"的温柔反讽 |

## 核心规则

1. **预测必须盲**：发布前完成预测，发布后补 = 数据污染
2. **Immutable 预测段**：上半段不可改，复盘只能追加
3. **盲打分隔离**：打分只读 `rubric_notes.md`，硬禁读 `rubric-memo.md`
4. **内容不卖课**：脚本中不出现"报名""限时优惠"类话术
5. **每条视频必有评论区钩子**：引导互动不卖课
6. **Buffer 警戒**：🔴<1天断更 │ 🟠1-2天偏低 │ 🟢3-5天健康 │ 🔵>5天积压
7. **样本数限制**：N=1 记录观察 │ N=5-9 首次 bump │ N≥20 回归反推
