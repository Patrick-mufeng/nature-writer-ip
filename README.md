# 🌿 自然写作IP — 短视频运营专家 Skill

> 带孩子走进自然，唤醒感受力，让文字从心流出。

## 简介

**nature-writer-ip** 是一个面向自然教育类短视频创作者的 **Reasonix Skill**，覆盖选题→脚本→预测→拍摄→发布→咨询引导→复盘→rubric升级全链路。

| 项目 | 内容 |
|---|---|
| **人设** | 自然里的写作老师 / 感受力唤醒师 / 不教技巧只教感受 |
| **目标人群** | 20-45岁女性（妈妈、教育工作者） |
| **平台** | 视频号 + 抖音 |
| **运行方式** | Reasonix subagent（`run_as: subagent`） |
| **模型** | deepseek-v4-pro |

---

## Skill 框架

整个 Skill 由三条线构成：

```
┌──────────────────────────────────────────────────────────────┐
│                      内容生产线                                │
│  seed → script → score → predict → shoot → publish           │
│  (选题) (脚本) (打分) (盲预测) (拍摄) (发布)                   │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                      运营变现线                                │
│  lead → retro → bump                                         │
│  (咨询引导) (数据复盘) (rubric升级)                            │
└──────────────────────────────────────────────────────────────┘
```

### 12 个命令

| # | 命令 | 功能 |
|---|---|---|
| 1 | **init** | 初始化项目，5问 onboarding + 创建脚手架 |
| 2 | **seed** | 基于50条痛点库+季节日历生成分类选题，管理选题状态 |
| 3 | **script** | 输出完整视频脚本（分镜/逐字稿/标题/封面/CTA），用户可修改确认 |
| 4 | **score** | 7维 rubric 快速打分（仅控制台输出） |
| 5 | **predict** | 盲预测，写入 immutable 预测文件 |
| 6 | **shoot** | 登记拍摄，检测改稿差异 |
| 7 | **publish** | 登记发布，触发推荐链 |
| 8 | **lead** | 评论区咨询信号识别+引导，记录线索 |
| 9 | **retro** | T+3 数据复盘（含成交数据） |
| 10 | **status** | 状态看板 |
| 11 | **bump** | Rubric 升级（≥5样本后） |
| 12 | **learn** | 对标账号学习 |

## 工作流程

### 全流程

```
init → seed(分类选题库) → script(可修改确认) → predict → shoot → publish → lead → retro → bump
                                                                   ↑________________________↓
```

### 各步骤详解

#### ① init — 初始化

一次性配置。回答5个问题（内容形态/发布状态/发布频率/数据回收/对标账号），自动生成脚手架文件。

**产出**：`.nature-state.json`、`rubric_notes.md`、选题库、脚本目录、预测目录等

#### ② seed — 选题

参考 **短视频IP选题生成专家** 方法论，融合50条目标人群痛点库（教育焦虑/自然缺失/感受力退化/自我成长）+ 季节日历。

**产出**：写入 `topic_library.md`，5大分类管理

**规则**：
- ✅ 已用选题自动标记，重复选择时提醒
- 🔄 一个选题发布后自动推荐下一个
- 📌 状态流转：可用 → 拍摄中 → 已用

#### ③ script — 脚本

参考 **短视频IP脚本生成专家** 方法论，按以下结构输出完整脚本：

1. 脚本概览
2. 四段式视频脚本（黄金钩子→痛点→价值→结尾）
3. 分镜建议（10个镜头）
4. 口播逐字稿（含语气标记）
5. 标题备选（8-10个）
6. 封面提示词（3个）
7. 视频简介 + 话题标签
8. CTA建议
9. ⭐ **评论区引导钩子**（必选，引导互动不卖课）

**关键环节**：脚本生成 → 用户修改确认 → 确认后才进入下一步。

#### ④ score — 快速打分（可选）

对脚本做 rubric 7维评估，仅控制台输出。

#### ⑤ predict — 盲预测

**核心纪律**：发布前必须预测，发布后补预测 = 数据污染 ❌

流程：隔离子流程打分 → 对比直觉 → 分歧≥2弹用户裁定 → 写入 immutable 预测文件

**产出**：`predictions/<id>.md`（预测段不可修改，复盘段可追加）

#### ⑥ shoot — 拍摄登记

验证 prediction 存在 → 检测实际稿 vs 脚本一致性（≥30%差异触发 v2 预测）

#### ⑦ publish — 发布登记

出队 → 标记 T+3 待复盘 → 更新日历 → 触发选题推荐链

#### ⑧ lead — 咨询引导 ⭐

**原则**：内容保持纯净，不出现硬广，不制造紧迫感。

| 时间 | 动作 |
|---|---|
| 发布后 2h | 刷评论区，标记咨询信号 |
| 发布后 24h | 对有信号的评论逐一回复 |
| 发布后 72h | 确认意向，写入 `leads.md` |

**信号识别**：出现"怎么教的""我孩子也这样""可以咨询你吗"等关键词时标记

**回复原则**：温暖不push，先帮后转化

#### ⑨ retro — 数据复盘

T+3天后自动触发，对比预测 vs 实绩 + **成交数据复盘**。

**产出**：
- 播放/点赞/评论/收藏/分享 + 派生比率
- 偏差分析
- **成交数据**：咨询信号数、已私信、已转化、收入
- **内容类型×转化率**跨视频统计
- Bump 触发评估

#### ⑩ status — 状态看板

实时显示 buffer 状态、待办列表、校准进度、库存规模。

#### ⑪ bump — Rubric 升级

6步强制流程：门槛检查 → 新公式 → 全量重打分 → 排序一致性 → 跨模型审核 → 落地

#### ⑫ learn — 对标账号学习

导入对标账号样本，分析 rubric 特征，更新 benchmark。

## IP 人设语气规范

| 风格 | 适用场景 | 语气 |
|---|---|---|
| A 温柔治愈 | 自然观察、亲子时光 | 像林间微风，有画面感有留白 |
| B 犀利扎心 | 教育焦虑、内卷批判 | 温柔刀，扎完给糖，不制造对立 |
| C 故事叙事 | 真实案例、孩子转变 | 起承转合，有细节有落点 |
| D 混合（默认） | 多数选题 | 故事为骨架，转折插观点，温柔收尾 |

## 变现逻辑

**不卖课，走1v1咨询路线**。

```
内容（纯净）→ 评论区钩子 → 评论互动 → 信号识别 → leads.md
                                                      ↓
                                              私信沟通 → 1v1咨询 → 成交
```

- 前30天只做信任，30天后开始引导咨询
- 不主动私信用户，只在用户公开提问时回复
- 回复先给一个免费建议，不直接报价

## Rubric 评分维度（v0）

7维等权，公式：`(ER×1.5 + SR×1.5 + HP×1.5 + QL + NA + AB + SAT) / 8.5 × 2.0`

| 维度 | 含义 | 自然教育场景解释 |
|---|---|---|
| ER (情感共鸣) | 情绪连接强度 | 妈妈看了想带孩子出门/想起自己童年 |
| SR (社会议题) | 教育焦虑共振 | 反内卷、反技巧崇拜、回归本真 |
| HP (钩子强度) | 前3秒抓眼 | 一个画面/一个问题让人停滑 |
| QL (金句密度) | 可传播句子 | "感受力是写作的第一支笔" |
| NA (叙事性) | 故事弧线 | 从"看见"到"感受"到"文字"的流动 |
| AB (受众广度) | 妈妈圈普适 | 不仅写作妈妈，所有关心孩子的女性 |
| SAT (讽刺深度) | 反讽/批判 | 对"培训班思维"的温柔反讽 |

## 核心规则

1. **预测必须盲**：发布前完成预测，发布后补 = 数据污染
2. **Immutable 预测段**：上半段不可改，复盘只能追加
3. **盲打分隔离**：打分只读 `rubric_notes.md`，硬禁读 `rubric-memo.md`
4. **内容不卖课**：脚本中不出现"报名""限时优惠"类话术
5. **每条视频必有评论区钩子**：引导互动不卖课
6. **Buffer 警戒**：🔴<1天断更 │ 🟠1-2天偏低 │ 🟢3-5天健康 │ 🔵>5天积压
