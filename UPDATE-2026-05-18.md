# nature-writer-ip 更新日志 (2026-05-18)

## 更新概述

基于 cheat-on-content 最新架构（GitHub 2026-05-18 版本）全面更新 nature-writer-ip skill。

## 主要变更

### 1. Schema 升级到 1.4

- **状态文件**: `.nature-state.json` 现在包含完整的 schema 1.4 字段
- **新增字段**: `schema_version`, `skill_version`, `typical_duration_seconds`, `rubric_form_mismatch`, `benchmark_name`, `calibration_samples_at_last_bump`, `last_prediction_self_scored`, `last_self_scored_at`, `consecutive_directional_errors`, `in_progress_session`
- **文件拆分**: `rubric_notes.md`（通用语言）+ `rubric-memo.md`（升级档案）

### 2. 新增模板文件

| 文件 | 说明 |
|------|------|
| `prediction.template.md` | 预测日志模板（7组件 + 复盘段） |
| `workflow.template.md` | 工作流速查文档 |
| `status.template.md` | 状态看板模板 |
| `rubric-memo.template.md` | Rubric升级Memo模板 |

### 3. SKILL.md 核心更新

- **/init**: 升级到 schema 1.4，新增 Phase 3/4 详细状态文件和 rubric 写入
- **/predict**: 新增 Phase 2.5 Disagreement Detection，完善 prediction 7组件格式
- **/shoot**: 新增 diff 计算逻辑（char_levenshtein_normalized），state.shoots 项 schema
- **/retro**: 新增复盘报告模板（实绩数据/评论关键词/验证推翻/观察记录/bump触发）
- **/bump**: 6步强制验证流程（Phase 0-6），新增 leak guard 自检
- **Key Rules**: 新增 blind channel 隔离规则、样本数→允许动作映射
- **Refusals**: 新增 3 条拒绝场景（rubric-memo 泄漏、N=3 回归等）

### 4. starter-rubric.md 重构

- **通用语言**: 移除所有实绩数据，只保留公式/维度定义/bucket边界
- **观察记录**: 新增模板格式，支持每次复盘后追加
- **重大跨视频观察**: 新增区域（≥2样本支持才升格）
- **被拒升级 log**: 新增区域

### 5. init_project.py 更新

- 兼容 schema 1.4
- 新增 `.cheat-cache/` 和 `.cheat-hooks/` 目录创建
- 新增模板文件复制（workflow, status, rubric-memo）
- 新增 `.gitignore` 创建
