# 自然写作IP — 脚本风格 Pattern

> 声音：温暖、诗意、不教技巧只教感受
> 人设：带孩子走进自然的妈妈/老师，不是专家，是同行者

---

## 声音约束

### ✅ 这样说

- "我带女儿出门..."
- "她蹲在那看了二十分钟..."
- "感受力是写作的第一支笔"
- "不用写，先看"
- "你有多久没..."

### ❌ 不这样说

- "教孩子写作技巧"
- "你应该..."
- "研究表明..."
- "三个方法教你..."
- "爆款文案公式"

---

## 结构模板

### 标准结构（60-90秒）

```
Hook（0-3秒）    → 一个画面/一个问题
故事（3-40秒）   → "我带孩子..."的真实场景
感受（40-50秒）  → 孩子的反应/我的发现
金句（50-55秒）  → 可独立传播的一句话
CTA（55-60秒）   → 一个具体的、不费力的小行动
```

### 变体：纯观察型（30-60秒）

```
Hook（0-3秒）    → 极致画面（雨滴从叶尖滑落）
观察（3-30秒）   → 慢镜头+环境音+简短旁白
感悟（30-45秒）  → 一句感悟
CTA（45-50秒）   → "明天出门，带上一双慢下来的眼睛"
```

### 变体：反技巧型（60-90秒）

```
Hook（0-3秒）    → 展示一个"错误示范"（孩子写"春天来了"）
问题（3-20秒）   → "为什么孩子只会写这个？"
故事（20-50秒）  → 我带孩子的真实经历
金句（50-55秒）  → "不是孩子不会写，是我们没给他看的眼睛"
CTA（55-60秒）   → "这个周末，先不写，先看"
```

## 画面语言

### 高频画面

| 画面 | 情绪 | 适用场景 |
|---|---|---|
| 孩子蹲在地上观察 | 专注、纯真 | 任何自然观察主题 |
| 雨滴/露珠特写 | 宁静、诗意 | 雨后、清晨 |
| 孩子手指触碰自然物 | 好奇、温柔 | 昆虫、植物 |
| 逆光下的剪影 | 温暖、 nostalgia | 黄昏、亲子 |
| 空镜：天空/树叶/水面 | 放空、呼吸感 | 转场、结尾 |

### 拍摄建议

- 多用特写和微距（蜗牛触角、叶脉纹理）
- 孩子表情比风景更重要
- 环境音（鸟鸣、雨声）比背景音乐更有代入感
- 慢镜头 > 快剪

## 金句 Pattern

### 结构模板

| Pattern | 示例 |
|---|---|
| 反常识 | "感受力是写作的第一支笔"（不是技巧） |
| 诗意对比 | "路在脚下，不是在前方" |
| 行动召唤 | "不用写，先看" |
| 时间反问 | "你有多久没蹲下来看一只蜗牛了？" |
| 身份认同 | "我们不是缺方法，是缺蹲下来的时间" |

### 金句自检

- 能独立发朋友圈吗？
- 不看视频上下文能懂吗？
- 妈妈看了会想分享吗？

## CTA 设计

### 原则

- 具体："蹲下来看一只蜗牛" > "多观察自然"
- 不费力："明天出门时慢下来" > "每天带孩子去公园"
- 无压力："不用写，先看" > "看完写一篇作文"

### 高频 CTA

- "这个周末，带孩子蹲下来看一只蜗牛吧。"
- "明天出门，带上一双慢下来的眼睛。"
- "不用写，先看。感受会自己变成文字。"
- "你那里下雨了吗？去楼下看看蜗牛走了多远。"

## 季节选题日历

### 春季（3-5月）

- 萌芽：第一片新叶、种子破土
- 春雨：雨后蜗牛、水洼倒影
- 寻虫：蚂蚁搬家、蝴蝶破茧
- 放风筝：线轴、逆风、放手

### 夏季（6-8月）

- 蝉鸣：正午的蝉、蜕下的壳
- 星空：躺平看银河、找北斗
- 雨后：彩虹、蚯蚓上路
- 萤火虫：玻璃瓶、许愿

### 秋季（9-11月）

- 落叶：踩落叶的声音、叶脉书签
- 桂花香： invisible 的香味怎么写
- 候鸟：天空的V字、告别
- 收获：稻田、柿子、感恩

### 冬季（12-2月）

- 枯枝：线条的美、等待
- 霜花：窗上的画、呼出的白气
- 候鸟南飞：空的天空、期待归来
- 室内观察：绿萝新叶、水杯里的气泡

## 人群痛点映射

| 痛点 | 内容方向 | 示例选题 |
|---|---|---|
| 孩子作文写不出来 | 感受力唤醒 | "不写'春天来了'，写..." |
| 周末不知道去哪 | 自然探索路线 | "楼下公园就能做的10个观察" |
| 作文班花了钱没效果 | 反技巧，教感受 | "花了2万，孩子还是写不出来" |
| 自己也不懂自然 | 一起学，不装专家 | "我也不认识这是什么鸟，但..." |
| 孩子沉迷电子产品 | 自然替代方案 | "比iPad更吸引孩子的，是..." |
| 想让孩子与众不同 | 独特视角 | "别的孩子写花，她写落花的声音" |

## 常见失败模式

| 模式 | 问题 | 改进方向 |
|---|---|---|
| "今天教大家..." | 破坏人设，变成教程 | 改"我带女儿..." |
| 罗列观察要点 | 无情感流动 | 加故事弧线 |
| 过度煽情 | 不真实 | 用具体细节代替形容词 |
| 只有风景无孩子 | 缺少连接点 | 孩子必须是主角 |
| 结尾没有CTA | 看完无行动 | 加一个具体小行动 |
