# 自然写作IP — 脚本模板

> 可直接套用的脚本结构模板

---

## 模板A：自然观察型（60-90秒）

### 适用
雨后观察、季节变化、昆虫植物

### 结构

```markdown
# [标题]

## Hook（0-3秒）
[画面：具体特写]
"[一个画面感极强的问题/陈述]"

## 故事（3-40秒）
[画面：场景展开]
"昨天[天气/场景]，我带[孩子]出门。

她[做了什么]，看了[多久]。

不是看我，是看[自然物]。

[自然物的细节]，[孩子的反应/问题]。

我说：[一个温柔的回答/感悟]。"

## 金句（40-50秒）
"[一句可独立传播的话]"

## CTA（50-60秒）
"[一个具体的小行动邀请]"

## 画面建议
- [3-5个具体画面描述]

## 音乐建议
- [音乐风格]

## 字幕风格
- [字体/颜色建议]
```

### 示例填充

```markdown
# 雨后带孩子找蜗牛

## Hook（0-3秒）
[画面：蹲下来，孩子手指轻触蜗牛触角]
"你有多久没蹲下来，看一只蜗牛走路了？"

## 故事（3-40秒）
[画面：雨后草地，蜗牛爬过叶片的痕迹]
"昨天雨后，我带女儿出门。她蹲在那片草地上，看了整整二十分钟。

不是看我，是看一只蜗牛。

它爬过的地方，会留下一条亮亮的线。女儿问我：妈妈，这是它的脚印吗？

我说：这是它的路。它不用知道要去哪，每走一步，路就在脚下。"

## 金句（40-50秒）
"感受力，是写作的第一支笔。不是技巧，是先看见。"

## CTA（50-60秒）
"这个周末，带孩子蹲下来看一只蜗牛吧。不用写，先看。"

## 画面建议
- 特写：蜗牛触角收缩
- 中景：孩子蹲姿+专注表情
- 空镜：雨后草地光斑
- 微距：蜗牛爬过的亮线

## 音乐建议
- 钢琴轻音乐，雨声环境音

## 字幕风格
- 手写体，温暖色调，米白底深棕字
```

## 模板B：反技巧型（60-90秒）

### 适用
教育理念、写作引导、反内卷

### 结构

```markdown
# [标题]

## Hook（0-3秒）
[画面：展示一个"错误示范"或普遍现象]
"[一个让人点头的问题]"

## 问题（3-20秒）
[画面：孩子困惑的表情/作业本]
"[描述问题：为什么孩子只会写...？]"

## 故事（20-50秒）
[画面：真实场景]
"我带[孩子][做了什么]。

[具体细节]。

[孩子的反应]。

[我的发现/感悟]。"

## 金句（50-55秒）
"[反常识的一句话]"

## CTA（55-60秒）
"[无压力的行动邀请]"
```

### 示例填充

```markdown
# 不写"春天来了"，写...

## Hook（0-3秒）
[画面：孩子作业本，上面写着"春天来了"]
"为什么孩子写春天，只会写'春天来了'？"

## 问题（3-20秒）
[画面：各种"春天来了"的作业本拼贴]
"我们教了那么多技巧，比喻、拟人、排比...

但孩子面对春天，还是只会写这四个字。"

## 故事（20-50秒）
[画面：带孩子出门，蹲在花前]
"上周，我带女儿去看樱花。

她没有写'春天来了'。

她写：'花瓣掉在水里，像小鱼在游泳。'

我问她：谁教你的？

她说：没人教，我就是看见了。"

## 金句（50-55秒）
"不是孩子不会写，是我们没给他看的眼睛。"

## CTA（55-60秒）
"这个周末，先不写，先看。感受会自己变成文字。"
```

## 模板C：纯感受型（30-60秒）

### 适用
极致画面、诗意表达、情绪短片

### 结构

```markdown
# [标题]

## Hook（0-3秒）
[画面：极致特写/慢镜头]
"[一句诗性的话，或沉默+环境音]"

## 观察（3-30秒）
[画面：慢镜头+环境音]
[无旁白或极简旁白，让画面说话]

## 感悟（30-45秒）
[画面：孩子或自己的反应]
"[一句感悟]"

## CTA（45-50秒）
"[温柔的邀请]"
```

### 示例填充

```markdown
# 雨滴从叶尖滑落

## Hook（0-3秒）
[画面：慢镜头，雨滴在叶尖聚集]
（无旁白，只有雨声）

## 观察（3-30秒）
[画面：雨滴越来越大，叶尖弯曲]
（雨声+远处鸟鸣）

## 感悟（30-45秒）
[画面：孩子伸手接雨滴]
"她说：雨滴在叶子上玩滑滑梯。"

## CTA（45-50秒）
"明天出门，带上一双慢下来的眼睛。"
```

## 模板D：季节回顾型（60-90秒）

### 适用
月末/季末/年末总结

### 结构

```markdown
# [标题]

## Hook（0-3秒）
[画面：季节标志性画面混剪]
"[时间感的问题]"

## 回顾（3-50秒）
[画面：该季节拍过的素材混剪]
"这个月，我们一起[做了什么]。

[场景1]。

[场景2]。

[场景3]。"

## 金句（50-55秒）
"[总结性的一句话]"

## CTA（55-60秒）
"[对未来的邀请]"
```

## 模板E：互动挑战型（60-90秒）

### 适用
发起话题、增加评论互动

### 结构

```markdown
# [标题]

## Hook（0-3秒）
"[一个挑战/邀请]"

## 示范（3-30秒）
"我[做了什么]。

[结果/发现]。"

## 邀请（30-50秒）
"[邀请观众做同样的事]"

## 金句（50-55秒）
"[一句话]"

## CTA（55-60秒）
"评论区告诉我[具体问题]"
```

### 示例填充

```markdown
# 挑战：今天出门，找到5种不同的绿色

## Hook（0-3秒）
"敢不敢挑战？今天出门，找到5种不同的绿色。"

## 示范（3-30秒）
"我试了。

草地的绿，叶子的绿，苔藓的绿，湖水的绿，还有...

我女儿说：妈妈，你的眼睛里也有绿色。"

## 邀请（30-50秒）
"你那里有什么绿？

深绿、浅绿、黄绿、蓝绿...

原来绿色不止一种，就像春天不止一个样子。"

## 金句（50-55秒）
"当我们开始分辨绿色，春天才真正开始。"

## CTA（55-60秒）
"评论区告诉我，你找到了几种绿？"
```

| 场景 | 推荐模板 | 时长 |
|---|---|---|
| 雨后/季节观察 | A | 60-90s |
| 教育理念/反内卷 | B | 60-90s |
| 极致画面/诗意 | C | 30-60s |
| 月末/季末总结 | D | 60-90s |
| 发起互动/挑战 | E | 60-90s |
